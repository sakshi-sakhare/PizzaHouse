PREREQUISITES:
1. Java 11+ installed
2. Maven 3.8+ installed
3. MySQL Server 8.0+ running
4. Tomcat 10.1+ installed

STEP 1: Setup Database
→ Create pizzahouse_db database
→ Run SQL scripts from BUILD_INSTRUCTIONS.txt or README.md
→ Configure credentials in src/main/java/com/pizzahouse/dao/DBConnection.java

STEP 2: Build Project
→ Windows: Run build.bat
→ Linux/Mac: Run build.sh
→ Manual: mvn clean package -DskipTests

STEP 3: Deploy to Tomcat
→ Copy target/pizzahouse.war to $CATALINA_HOME/webapps/
→ Start Tomcat
→ Access at http://localhost:8080/pizzahouse
